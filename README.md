## Cinema Booking System

A full-stack Cinema Booking System developed to simplify the movie ticket reservation process for both customers and cinema staff. The application allows users to browse available movies, filter films by category and language, search for showtimes, select seats through an interactive seat map, and book tickets online with real-time seat availability.

The system dynamically updates booked seats to prevent duplicate reservations and provides an intuitive booking workflow from movie selection to checkout. After a successful reservation, customers automatically receive a booking confirmation email containing their ticket details. A contact form is also integrated, allowing users to communicate directly with the cinema.

To enhance performance and demonstrate data structure concepts, the system utilizes a **Binary Search Tree (BST)** to organize and display movie showtimes chronologically, while a **Queue (Linked List)** manages upcoming movie announcements efficiently. Database transactions and server-side validation ensure booking consistency and prevent double-booking during concurrent reservations.

### Key Features

* Browse and search available movies
* Filter movies by category and language
* Interactive seat selection with real-time availability
* Online movie ticket reservation
* Dynamic showtime scheduling
* Email booking confirmation
* Upcoming movie announcements
* Contact form with email integration
* Server-side booking validation
* Responsive user interface

**Tech Stack:** PHP, MySQL, JavaScript, HTML5, CSS3, Bootstrap, AJAX, PHPMailer

**Data Structures & Algorithms:**

* Binary Search Tree (BST) for chronological showtime management.
* Queue (Linked List) for upcoming movie listings.
* Dynamic filtering and searching algorithms for movie discovery.
